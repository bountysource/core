How Bounties Work

Bounties are cash rewards on specific bugs or feature requests that encourage development.

1
Create a bounty.

  • Get bugs fixed and new features added sooner.
  • A little bit goes a long way with crowdfunding.
  • Attract new developers and grow the community.

2
Earn money by solving issues.

  • Submit your pull request to the project.
  • Let Bountysource know by creating a solution.
  • Get paid when your code is merged.

3
GitHub, Trac, Bugzilla, and more.

We integrate closely with all issues trackers.
